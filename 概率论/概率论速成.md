## 第一章 随机事件及其概率

### 1.1 随机试验与基本概念

| 概念 | 定义 | 记法 |
|------|------|------|
| **随机试验 E** | 满足三个特征：① 可在相同条件下**重复**进行；② 所有可能结果**已知**；③ 每次试验结果**不确定** | $E$ |
| **样本空间** | 随机试验 E 的所有可能结果构成的集合 | $\Omega$ 或 $S$ |
| **样本点** | 样本空间中的每一个元素 | $e$ 或 $\omega$ |
| **随机事件** | 样本空间的子集。事件发生 = 子集中**某一个**样本点出现（不是全部） | $A, B, C\ldots$ |
| **必然事件** | 必然发生的事件 | $\Omega$ |
| **不可能事件** | 不可能发生的事件 | $\varnothing$ |

### 1.2 事件间的关系与运算

| 关系/运算        | 符号                 | 含义                                | 关键记忆                      |
| ------------ | ------------------ | --------------------------------- | ------------------------- |
| **包含**       | $A \subset B$      | A 发生必然导致 B 发生                     | —                         |
| **和（并）**     | $A \cup B$         | A、B **至少发生一个**                    | —                         |
| **积（交）**     | $A \cap B$ 或 $AB$  | A、B **同时发生**                      | —                         |
| **差**        | $A - B$            | A 发生**且** B 不发生                   | $A - B = A - AB$，「减谁谁不发生」 |
| **互斥（互不相容）** | $AB = \varnothing$ | A 与 B 不能同时发生                      | —                         |
| **对立事件** | $\bar{A}$（或 $A^c$） | 非 A，满足 $\bar{A} \cup A = \Omega$、$\bar{A} \cap A = \varnothing$ | $\bar{\bar{A}} = A$ |

#### 运算法则

| 法则       | 公式                                                                                        |
| -------- | ----------------------------------------------------------------------------------------- |
| **交换律**  | $A \cup B = B \cup A$，$A \cap B = B \cap A$                                               |
| **结合律**  | $(A \cup B) \cup C = A \cup (B \cup C)$                                                   |
| **分配律**  | $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$                                          |
| **德摩根律** | $\overline{A \cup B} = \bar{A} \cap \bar{B}$，$\overline{A \cap B} = \bar{A} \cup \bar{B}$ |

> 🔑 **德摩根律口诀**：「长杠变短杠，开口变方向」

### 1.3 概率的定义与性质

| 性质 | 公式 | 说明 |
|------|------|------|
| **非负性** | $0 \leq P(A) \leq 1$ | 概率值域在 $[0,1]$ |
| **规范性** | $P(\Omega) = 1$，$P(\varnothing) = 0$ | 必然事件概率为 1 |
| **有限可加性** | 若 $AB = \varnothing$，则 $P(A \cup B) = P(A) + P(B)$ | 互斥时直接相加 |
| **互补性** | $P(\bar{A}) = 1 - P(A)$ | — |
| **减法公式** | $P(A - B) = P(A) - P(AB)$ | 若 $B \subset A$，则 $= P(A) - P(B)$ |
| **加法公式（两项）** | $P(A \cup B) = P(A) + P(B) - P(AB)$ | — |
| **加法公式（三项）** | $P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)$ | 🔑 **加奇减偶** |

> 🔑 **加法公式规律**：「加奇减偶」— A∪B（2项，偶数）减；A∪B∪C（3项，奇数）加 ABC
>
> **减法公式推广**：$P(\square - \triangle) = P(\square) - P(\square \cap \triangle)$（前提是 $\square \cap \triangle \subseteq \square$）
> 如 $P(AB - C) = P(AB) - P(ABC)$；$P(A\bar{B}) = P(A) - P(AB)$

#### 📝 例题 1-3：概率性质计算

**例 1.1** 已知 $P(A)=0.5$，$P(A+B)=0.8$，且 $A$ 与 $B$ 互不相容，求 $P(B)$。

**解**：加法公式 $P(A+B) = P(A) + P(B) - P(AB)$。互不相容 $\implies AB = \varnothing \implies P(AB)=0$。
$0.8 = 0.5 + P(B) - 0 \implies P(B) = 0.3$。

**例 1.2**（考研真题）已知 $P(A)=P(B)=P(C)=\frac{1}{4}$，$P(AB)=0$，$P(BC)=0$，$P(AC)=\frac{1}{8}$，求 A、B、C 至少发生一个的概率。

**解**：由三项加法公式「**加奇减偶**」：
$$P(A+B+C) = \frac{1}{4}+\frac{1}{4}+\frac{1}{4} - 0 - \frac{1}{8} - 0 + P(ABC)$$

关键：$ABC \subseteq AB$，故 $P(ABC) \le P(AB) = 0$。又 $P(ABC) \ge 0$（非负性），由夹逼得 $P(ABC)=0$。
$$P(A+B+C) = \frac{5}{8}$$

> 🔑 子集概率 $\le$ 父集概率：若 $X \subseteq Y$，则 $P(X) \le P(Y)$。

**例 1.3** 已知 $P(A)=0.7$，$P(A-B)=0.3$，求 $P(\overline{AB})$。

**解**：先用减法公式求 $P(AB)$：$P(A-B) = P(A) - P(AB) \implies 0.3 = 0.7 - P(AB) \implies P(AB) = 0.4$。

再求 $\overline{AB}$（整体拔）的概率：$P(\overline{AB}) = 1 - P(AB) = 1 - 0.4 = 0.6$。

### 1.4 条件概率与乘法公式

$$P(B \mid A) = \frac{P(AB)}{P(A)}, \quad P(A) > 0$$

**乘法公式**：$P(AB) = P(A) \cdot P(B \mid A)$，$P(ABC) = P(A) \cdot P(B \mid A) \cdot P(C \mid AB)$

#### 📝 例 1.4：条件概率

已知 $P(A+B)=0.5$，$P(B)=0.4$，求 $P(A \mid \bar{B})$。

**解**：
加法公式：$0.5 = P(A) + 0.4 - P(AB) \implies P(A) - P(AB) = 0.1$

条件概率公式：
$$P(A \mid \bar{B}) = \frac{P(A\bar{B})}{P(\bar{B})} = \frac{P(A) - P(AB)}{1 - 0.4} = \frac{0.1}{0.6} = \frac{1}{6}$$

> 💡 看到 $A\bar{B}$ 就用减法公式：$P(A\bar{B}) = P(A) - P(AB)$。

### 1.5 独立性

**定义**：$P(AB) = P(A) \cdot P(B)$。等价：$P(B \mid A) = P(B)$，$P(B \mid A) = P(B \mid \bar{A})$。

> 🔑 **「有拔无拔对独立没有影响」**：$A$ 与 $B$ 独立 $\implies$ $A$ 与 $\bar{B}$、$\bar{A}$ 与 $B$、$\bar{A}$ 与 $\bar{B}$ 都相互独立。

**三事件独立**须同时满足：两两独立 + $P(ABC)=P(A)P(B)P(C)$。

#### 📝 例 1.5：独立性——两种解法

已知 $A$ 与 $B$ 独立，$P(A+B)=0.5$，$P(B)=0.4$，求 $P(\bar{B} \mid A)$。

**解法一（常规）**：由独立 $P(AB)=0.4P(A)$，加法公式 $P(A)+0.4-0.4P(A)=0.5 \implies P(A)=\frac{1}{6}$。
$$P(\bar{B} \mid A) = \frac{P(A\bar{B})}{P(A)} = \frac{P(A)-0.4P(A)}{P(A)} = 0.6$$

**解法二（秒杀）**：独立 $\implies$ $A$ 与 $\bar{B}$ 也独立，$\therefore P(\bar{B} \mid A) = P(\bar{B}) = 1-0.4 = 0.6$。

### 1.6 古典概型 ⭐

**条件**：有限性 + 等可能性。核心公式：$P(A) = \dfrac{A\text{ 的样本点个数}}{\Omega\text{ 的样本点个数}}$

排列 $A_n^m = \frac{n!}{(n-m)!}$（讲顺序）；组合 $C_n^m = \frac{n!}{m!(n-m)!}$（不讲顺序）。

#### 📝 例 1.6：取球问题

袋中有 5 白 6 黑，任取两球。求：(1) 一黑一白的概率；(2) 至少一只黑球的概率。

**法一（不计顺序）**：$|\Omega| = C_{11}^2 = 55$
(1) $|A| = C_5^1 \cdot C_6^1 = 30 \implies P(A) = \frac{30}{55} = \frac{6}{11}$
(2) 对立事件（全白）：$P(\bar{B}) = \frac{C_5^2}{55} = \frac{10}{55} = \frac{2}{11} \implies P(B) = \frac{9}{11}$

> 🔑 **「分步要相乘，分类要相加」**；**「至少」问题优先考虑对立事件**。

### 1.7 全概率公式 ⭐

**完备事件组（划分）**：两两互斥 + 和为全集。划分方式**不唯一**。

$$P(B) = \sum_{i=1}^{n} P(A_i) \cdot P(B \mid A_i)$$

### 1.8 贝叶斯公式 ⭐

$$P(A_i \mid B) = \frac{P(A_i) \cdot P(B \mid A_i)}{\sum_{j} P(A_j) \cdot P(B \mid A_j)}$$

> 💡 **全概率 = 由因求果**；**贝叶斯 = 由果溯因**。看到「已知……条件下，问……」→ 贝叶斯。

#### 📝 例 1.7：两阶段摸球（全概率 + 贝叶斯）

甲袋 5 红 4 白，乙袋 4 红 5 白。先从甲取一球放入乙，再从乙取球。
(1) 求从乙取到白球的概率；(2) 若取到白球，求从甲取出的也是白球的概率。

**解**：设 $B_1$=甲取红，$B_2$=甲取白（完备事件组）；$A$=乙取白球（复杂事件）。

(1) 全概率：
$$P(A) = P(B_1)P(A|B_1) + P(B_2)P(A|B_2) = \frac{5}{9}\cdot\frac{5}{10} + \frac{4}{9}\cdot\frac{6}{10} = \frac{49}{90}$$

(2) 贝叶斯：
$$P(B_2|A) = \frac{P(B_2)P(A|B_2)}{P(A)} = \frac{\frac{4}{9}\cdot\frac{6}{10}}{\frac{49}{90}} = \frac{24}{49}$$

#### 📝 例 1.8：色盲检验（全概率 + 贝叶斯）

男性 5% 色盲，女性 0.25% 色盲。男女各半。任选一人恰好是色盲，求此人是男性的概率。

**解**：$B_1$=男性，$B_2$=女性；$A$=色盲。

全概率：$P(A) = \frac{1}{2}\cdot\frac{1}{20} + \frac{1}{2}\cdot\frac{1}{400} = \frac{21}{800}$

贝叶斯：$P(B_1|A) = \frac{\frac{1}{2}\cdot\frac{1}{20}}{\frac{21}{800}} = \frac{20}{21}$

> 🔑 **两阶段实验 → 全概率/贝叶斯**。先找完备事件组（第一阶段结果），再设复杂事件（第二阶段关心的结果）。

### 1.9 几何概型

$$P(A) = \frac{A \text{ 的长度（面积/体积）}}{\Omega \text{ 的长度（面积/体积）}}$$

> 💡 遇到「大于」「小于」不好画时 → 先画出等于的情况。

### 1.10 伯努利概型

伯努利试验：每次只有两种结果。$n$ 重伯努利：独立重复 $n$ 次，每次成功概率 $p$。

**二项概率公式**：$P_n(k) = C_n^k \cdot p^{\,k} \cdot (1-p)^{\,n-k}, \quad k = 0, 1, \ldots, n$

> 🔑 三要素：**独立** + **$n$ 次** + **两种结果**，三者都满足就用二项概率。

#### 📝 例 1.9：打靶

命中率 0.8，独立射击五次，求恰好命中两次的概率。

**解**：$n=5, p=0.8, k=2$
$$P_5(2) = C_5^2 \cdot 0.8^2 \cdot 0.2^3 = 10 \times 0.64 \times 0.008 = 0.0512$$

### 微积分预备：常用导数与积分公式

> 连续型随机变量涉及大量求导和积分，以下公式必须熟练。

**导数表**：

| $f(x)$ | $f'(x)$ | | $f(x)$ | $f'(x)$ |
|--------|---------|---|--------|---------|
| $C$ | $0$ | | $x^n$ | $nx^{\,n-1}$ |
| $e^x$ | $e^x$ | | $a^x$ | $a^x \ln a$ |
| $\ln x$ | $\frac{1}{x}$ | | $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ | | $\tan x$ | $\sec^2 x$ |

**不定积分表**：

| $\displaystyle\int f(x)\,dx$ | 结果 | | $\displaystyle\int f(x)\,dx$ | 结果 |
|------------------------------|------|---|------------------------------|------|
| $\int x^n\,dx$ | $\frac{x^{n+1}}{n+1}+C\;(n\neq -1)$ | | $\int \frac{1}{x}\,dx$ | $\ln\vert x\vert + C$ |
| $\int e^x\,dx$ | $e^x + C$ | | $\int a^x\,dx$ | $\frac{a^x}{\ln a}+C$ |
| $\int \sin x\,dx$ | $-\cos x + C$ | | $\int \cos x\,dx$ | $\sin x + C$ |

> 💡 连续型概率计算本质就是**求导**和**定积分**。规范性 $\int f(x)dx=1$ 用定积分求参数，分布函数 $F(x)=\int f(t)dt$ 用不定积分+规范性定常数。

## 第二章 一维随机变量及其分布

### 2.1 随机变量与分布函数

- **随机变量**：定义在 $\Omega$ 上的实值函数 $X = X(e)$，分 离散型、连续型、其他
- **分布函数**：$F(x) = P\{X \leq x\}$

**分布函数性质**：非负 $[0,1]$；$F(-\infty)=0, F(+\infty)=1$；单调不减；右连续。

**区间概率**：$P\{a < X \leq b\} = F(b) - F(a)$；$P\{X = a\} = F(a) - F(a-0)$；连续型 $P\{X=a\}=0$。

### 2.2 离散型随机变量

**分布律**：表格表示取值与概率。**八字口诀**：「先求取值，再求相应取值的概率。」

| 分布 | 分布律 | 参数 | 识别关键词 |
|------|--------|------|-----------|
| **0-1 分布** $B(1,p)$ | $P(X=1)=p,\;P(X=0)=1-p$ | $p$ | 两种结果 |
| **二项分布** $B(n,p)$ | $C_n^k p^k (1-p)^{n-k}$ | $n,p$ | 独立 + n次 + 两种结果 |
| **泊松分布** $P(\lambda)$ | $\frac{\lambda^k}{k!}e^{-\lambda}$ | $\lambda$ | 稀有事件 |
| **几何分布** $G(p)$ | $(1-p)^{k-1}p$ | $p$ | 首次成功 |

#### 📝 例 2.1：掷骰子求较小点数的分布率 ⭐

将骰子掷两次，$X$ 表示较小点数。求 $X$ 的分布律。

**解**：$X$ 取值 $\{1,2,3,4,5,6\}$。总情况数 $6 \times 6 = 36$（用排列 $A_6^1 \cdot A_6^1$，与次序有关）。

**表格法**：画 6×6 表，行=第一次点数，列=第二次点数，格子值为 $\min(\text{行},\text{列})$：

|  | **1** | **2** | **3** | **4** | **5** | **6** |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| **1** | 1 | 1 | 1 | 1 | 1 | 1 |
| **2** | 1 | 2 | 2 | 2 | 2 | 2 |
| **3** | 1 | 2 | 3 | 3 | 3 | 3 |
| **4** | 1 | 2 | 3 | 4 | 4 | 4 |
| **5** | 1 | 2 | 3 | 4 | 5 | 5 |
| **6** | 1 | 2 | 3 | 4 | 5 | 6 |

统计各值出现次数（L 形计数）：
- $X=1$：第 1 行(6) + 第 1 列(5，不含交叉点) = 11 → $\frac{11}{36}$
- $X=2$：第 2 行(5) + 第 2 列(4) = 9 → $\frac{9}{36}$
- $X=3=\frac{7}{36}$，$X=4=\frac{5}{36}$，$X=5=\frac{3}{36}$，$X=6=\frac{1}{36}$

验证：$11+9+7+5+3+1=36$ ✓

#### 📝 例 2.2：二项分布反求 p

四次独立射击，每次命中率 $p$。至少命中一次的概率为 $\frac{80}{81}$，求 $p$。

**解**：$X \sim B(4, p)$。对立事件：
$$P(\text{至少一次}) = 1 - P(X=0) = 1 - (1-p)^4 = \frac{80}{81}$$
$$(1-p)^4 = \frac{1}{81} = \left(\frac{1}{3}\right)^4 \implies 1-p = \frac{1}{3} \implies p = \frac{2}{3}$$

### 2.3 连续型随机变量

$F(x) = \int_{-\infty}^x f(t)\,dt$，$f(x) \geq 0$ 为密度函数。

**密度性质**：非负；$\int_{-\infty}^{+\infty} f(x)\,dx = 1$（规范性）；$P(a<X<b) = \int_a^b f(x)\,dx = F(b) - F(a)$。

> 💡 $f(x)$ 连续点处 $F'(x) = f(x)$。**不定积分秒杀法**：小 $f$ 的不定积分 = 大 $F$ + $C$，用规范性 $F(+\infty)=1$、$F(-\infty)=0$ 定 $C$。

#### 📝 例 2.3：已知 $F(x)$ 求 $f(x)$ 和概率

$$F(x) = \begin{cases} 0, & x < 1 \\ \ln x, & 1 \le x < e \\ 1, & x \ge e \end{cases}$$

求：(1) $f(x)$；(2) $P(X < 2)$；(3) $P(0 < X < 3)$（两种方法）。

**解**：
(1) $f(x) = F'(x) = \begin{cases} 1/x, & 1 < x < e \\ 0, & \text{其他} \end{cases}$

(2) $P(X<2) = P(X \le 2) = F(2) = \ln 2$（连续型加不加等号不影响）

(3) **法一**：$F(3)-F(0) = 1 - 0 = 1$
**法二**：$\int_0^3 f(x)dx = \int_1^e \frac{1}{x}dx = \ln e - \ln 1 = 1$（积分区间与有效区间取交集）

#### 📝 例 2.4：已知 $f(x)$ 含参，求 $k$ 和 $F(x)$（两分段·秒杀法）

$$f(x) = \begin{cases} k e^{-2x}, & x > 0 \\ 0, & x \le 0 \end{cases}$$

求：(1) $k$；(2) $F(x)$。

**解**：
(1) 规范性：$\int_0^{+\infty} k e^{-2x}dx = k \cdot \frac{1}{2} = 1 \implies k = 2$

(2) 不定积分秒杀法：
- $x > 0$：$F(x) = -e^{-2x} + C_1$（谁的导数是 $2e^{-2x}$？$\to -e^{-2x}$）
- $x \le 0$：$F(x) = C_2$
- $F(-\infty)=0 \implies C_2=0$；$F(+\infty)=1 \implies 0+C_1=1 \implies C_1=1$
$$F(x) = \begin{cases} 1 - e^{-2x}, & x > 0 \\ 0, & x \le 0 \end{cases}$$

#### 📝 例 2.5：已知 $f(x)$ 求 $F(x)$（三分段·秒杀法进阶）

$$f(x) = \begin{cases} 0, & x < 0 \\ 1, & 0 \le x < 1 \\ 0, & x \ge 1 \end{cases}$$

求 $F(x)$。

**解**：不定积分 + 规范性 + **分段点连续性**（三分段三未知数，需三个条件）：
$$F(x) = \begin{cases} C_1, & x < 0 \\ x + C_2, & 0 \le x < 1 \\ C_3, & x \ge 1 \end{cases}$$

$F(-\infty)=0 \implies C_1=0$；$F(+\infty)=1 \implies C_3=1$。
在 $x=0$ 连续：$\lim_{x\to 0^-}F(x) = C_1 = 0 = \lim_{x\to 0^+}F(x) = C_2 \implies C_2=0$
$$F(x) = \begin{cases} 0, & x < 0 \\ x, & 0 \le x < 1 \\ 1, & x \ge 1 \end{cases}$$

> 🔑 三分段需要**三个条件**确定常数：$F(-\infty)=0$ + $F(+\infty)=1$ + **分段点连续性**（左极限=右极限）。

### 2.4 三种常见连续型分布

| 分布                         | 密度 $f(x)$                                                     | 关键特征                             |
| -------------------------- | ------------------------------------------------------------- | -------------------------------- |
| **均匀分布** $U(a,b)$          | $\frac{1}{b-a},\; a<x<b$                                      | 子区间概率 $= \frac{d-c}{b-a}$        |
| **指数分布** $e(\lambda)$ | $\lambda e^{-\lambda x},\; x>0$ | $F(x)=1-e^{-\lambda x}$，**无记忆性** |
| **正态分布** $N(\mu,\sigma^2)$ | $\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ | 关于 $\mu$ 对称；$\sigma$ 越小越尖        |

**标准化**：$Z = \frac{X-\mu}{\sigma} \sim N(0,1)$。$\Phi(0)=0.5$；$\Phi(-x) = 1 - \Phi(x)$（对称性公式）。

**区间概率公式**：$P(a < X < b) = \Phi\!\left(\frac{b-\mu}{\sigma}\right) - \Phi\!\left(\frac{a-\mu}{\sigma}\right)$

**上 $\alpha$ 分位点** $u_\alpha$：$P(X > u_\alpha) = \alpha$。

##### Gamma 分布（指数 & 卡方的母分布）

$$f(x) = \frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\lambda x},\; x \ge 0,\; \alpha>0,\; \lambda>0$$

| 性质 | 内容 |
|------|------|
| **$\Gamma$ 函数** | $\Gamma(\alpha)=\int_0^{+\infty}x^{\alpha-1}e^{-x}dx$，$\Gamma(1)=1$，$\Gamma(\frac{1}{2})=\sqrt{\pi}$ |
| **递推** | $\Gamma(\alpha+1)=\alpha\Gamma(\alpha)$；$\alpha$ 为正整数时 $\Gamma(\alpha)=(\alpha-1)!$ |
| **退化 $\alpha=1$** | $\Gamma(1,\lambda)$ = **指数分布** $e(\lambda)$ |
| **退化 $\alpha=\frac{n}{2},\lambda=\frac{1}{2}$** | $\Gamma(\frac{n}{2},\frac{1}{2})$ = **卡方分布** $\chi^2(n)$ |
| **期望与方差** | $E(X)=\frac{\alpha}{\lambda}$，$D(X)=\frac{\alpha}{\lambda^2}$ |

#### 📝 例 2.6：均匀分布 + 二次方程有实根

$X \sim U(0, 5)$，求关于 $t$ 的方程 $4t^2 - 4Xt + (X+2) = 0$ 有实根的概率。

**解**：判别式 $\Delta = 16X^2 - 16(X+2) = 16(X^2 - X - 2) \ge 0$
$(X-2)(X+1) \ge 0 \implies X \ge 2 \text{ 或 } X \le -1$

$$P = P(X \ge 2) + P(X \le -1) = \frac{5-2}{5-0} + 0 = \frac{3}{5}$$

#### 📝 例 2.7：正态分布对称性

$X \sim N(2, \sigma^2)$，$P(2<X<4)=0.3$，求 $P(X<0)$。

**解**：
**法一（图示，推荐）**：$\mu=2$ 为对称轴。$[2,4]$ 面积 = 0.3，对称区间 $[0,2]$ 面积也是 0.3。对称轴左侧总面积 = 0.5。$\therefore P(X<0) = 0.5 - 0.3 = 0.2$。

**法二（标准化）**：$P(0<Z<\frac{2}{\sigma})=0.3 \implies \Phi(\frac{2}{\sigma}) = 0.8$。
$P(X<0) = \Phi(-\frac{2}{\sigma}) = 1 - \Phi(\frac{2}{\sigma}) = 0.2$。

### 2.5 随机变量函数的分布

**离散型**：先求 $Y=g(X)$ 的取值（去重），再概率相加（同一 $Y$ 值对应的多个 $X$ 概率求和）。

**连续型**：**分布函数法**——先求 $F_Y(y) = P\{g(X) \le y\}$，再求导得 $f_Y(y)$。
1. 将 $X$ 的分段点代入 $g(X)$ → 得 $Y$ 的分段点
2. 分区间求积分，**积分区间要与 $f_X$ 有效区间取交集**
3. 求导得密度

#### 📝 例 2.8：离散型 $Y = X^2$

已知 $X$ 分布律：$P(X=-2)=\frac{1}{5}, P(X=-1)=\frac{1}{6}, P(X=0)=\frac{1}{5}, P(X=1)=\frac{1}{15}, P(X=3)=\frac{11}{30}$

求 $Y = X^2$ 的分布律。

**解**：取值：$(-2)^2=4, (-1)^2=1, 0^2=0, 1^2=1, 3^2=9$ → 去重得 $\{0,1,4,9\}$

概率：
- $P(Y=0) = P(X=0) = \frac{1}{5}$
- $P(Y=1) = P(X=-1)+P(X=1) = \frac{1}{6}+\frac{1}{15} = \frac{7}{30}$
- $P(Y=4) = P(X=-2) = \frac{1}{5}$
- $P(Y=9) = P(X=3) = \frac{11}{30}$

验证：$\frac{6}{30}+\frac{7}{30}+\frac{6}{30}+\frac{11}{30}=1$ ✓

#### 📝 例 2.9：连续型 $Y = X^3$（难点）

$X \sim U(0, 2)$，$Y = X^3$。求 $f_Y(y)$。

**解**：
1. $f_X(x) = \frac{1}{2},\; 0<x<2$。$X$ 分段点 0, 2 → 代入 $Y=X^3$ 得 $Y$ 分段点 0, 8。

2. $F_Y(y) = P(X^3 \le y) = P(X \le \sqrt[3]{y}) = \int_{-\infty}^{\sqrt[3]{y}} f_X(x)dx$
   - $y < 0$：积分区间与 $(0,2)$ 无交集 → $F_Y(y) = 0$
   - $0 \le y < 8$：$F_Y(y) = \int_0^{\sqrt[3]{y}} \frac{1}{2}dx = \frac{1}{2}\sqrt[3]{y}$
   - $y \ge 8$：积分覆盖全区域 → $F_Y(y) = 1$

3. 求导：
$$f_Y(y) = F_Y'(y) = \begin{cases} \frac{1}{6y^{2/3}}, & 0 < y < 8 \\ 0, & \text{其他} \end{cases}$$

> 🔑 端点处改变个别点不影响密度函数。

## 第三章 二维随机变量及其分布

### 3.1 基本概念

**联合分布函数**：$F(x,y) = P\{X \leq x, Y \leq y\}$

**离散型**：联合分布律表格。**八字口诀**：「先求取值，再求概率。」

**连续型**：$F(x,y) = \iint f(u,v)\,dudv$。规范性 $\iint f(x,y)dxdy=1$。

### 3.2 边缘分布

| | 离散型 | 连续型 |
|---|--------|--------|
| $X$ 边缘 | $p_{i\cdot} = \sum_j p_{ij}$（行行相加） | $f_X(x) = \int f(x,y)\,dy$ |
| $Y$ 边缘 | $p_{\cdot j} = \sum_i p_{ij}$（列列相加） | $f_Y(y) = \int f(x,y)\,dx$ |

> 💡 边缘 = 把另一维积掉（或加掉）。

### 3.3 条件分布

条件分布 = 联合 / 边缘：$P(X=x_i|Y=y_j) = p_{ij}/p_{\cdot j}$；$f_{X|Y}(x|y) = f(x,y)/f_Y(y)$（分母 $\neq 0$）。

### 3.4 独立性 ⭐

离散型独立 $\iff$ $p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$（等价：各行成比例）。连续型独立 $\iff$ $f(x,y) = f_X(x)f_Y(y)$。

### 3.5 两个随机变量函数的分布

**离散型**：解方程 $g(X,Y)=z$，将对应联合概率相加。

**连续型**：**分布函数法**——$F_Z(z) = P\{g(X,Y) \le z\} = \iint_{g(x,y)\le z} f(x,y)\,dxdy$，再求导。

**卷积公式**（$Z=X+Y$ 且独立）：$f_Z(z) = \int f_X(x)f_Y(z-x)\,dx$

> 🔑 分段点求法：将非零区域的**边角点**坐标代入 $z=g(x,y)$。

#### 📝 第三章典型例题

**例 3.1** 两封信投四个邮筒，$X$=1号筒信数，$Y$=2号筒信数。

求：(1) 联合分布律；(2) 边缘分布律；(3) $Y=1$ 时 $X$ 的条件分布律；(4) $X$ 与 $Y$ 是否独立？

**解**：
(1) 总情况 $4 \times 4 = 16$。联合分布律：

| X\Y | 0 | 1 | 2 |
|-----|-----|-----|------|
| 0 | $\frac{1}{4}$ | $\frac{1}{4}$ | $\frac{1}{16}$ |
| 1 | $\frac{1}{4}$ | $\frac{1}{8}$ | 0 |
| 2 | $\frac{1}{16}$ | 0 | 0 |

(2) 边缘分布：行行相加、列列相加。$X$ 边缘：$P(0)=\frac{9}{16}, P(1)=\frac{6}{16}, P(2)=\frac{1}{16}$。$Y$ 与 $X$ 对称相同。

(3) $Y=1$ 条件：$P(Y=1)=\frac{3}{8}$。$P(X=0|Y=1)=\frac{\frac{1}{4}}{\frac{3}{8}}=\frac{2}{3}$；$P(X=1|Y=1)=\frac{\frac{1}{8}}{\frac{3}{8}}=\frac{1}{3}$；$P(X=2|Y=1)=0$。

(4) 速判：第一行 $(\frac{1}{4}, \frac{1}{4}, \frac{1}{16})$ 与第二行 $(\frac{1}{4}, \frac{1}{8}, 0)$ 不成比例 → **不独立**。严格验证：$P(X=2,Y=2)=0 \neq \frac{1}{16}\times\frac{1}{16}$。

**例 3.2** 联合密度 $f(x,y) = \begin{cases} kx, & 0 < x < y < 1 \\ 0, & \text{其他} \end{cases}$

求：(1) $k$；(2) $P(X+Y<1)$；(3) 边缘密度；(4) $f_{Y|X}(y|x)$；(5) 独立性。

**解**：
(1) $\iint_D kx\,dxdy = 1$。先定 $x(0\to 1)$，后穿 $y(x\to 1)$：
   $\int_0^1 \int_x^1 kx\,dy\,dx = k\int_0^1 x(1-x)dx = \frac{k}{6} = 1 \implies k=6$

(2) $X+Y<1$ 交集区域：$x(0\to \frac{1}{2}), y(x\to 1-x)$
   $\int_0^{1/2}\int_x^{1-x} 6x\,dy\,dx = 6\int_0^{1/2} x(1-2x)dx = \frac{1}{4}$

(3) $f_X(x) = \int_x^1 6x\,dy = 6x(1-x),\; 0<x<1$
   $f_Y(y) = \int_0^y 6x\,dx = 3y^2,\; 0<y<1$

(4) $f_{Y|X}(y|x) = \frac{6x}{6x(1-x)} = \frac{1}{1-x},\; x<y<1$（分母 $0<x<1$ 时）

(5) $f_X(x)f_Y(y) = 6x(1-x) \cdot 3y^2 \neq 6x = f(x,y)$ → **不独立**。

> 🔑 二重积分定限口诀：**先定 $x$ 后穿 $y$**（用平行于 $y$ 轴的射线穿过阴影区域）。

**例 3.3** 已知 $(X,Y)$ 联合分布律，求 $Z = X+Y$ 分布律。

**口诀**：先求取值（$X+Y$ 的所有可能和，去重），再求概率（同一 $Z$ 值对应多组 $(X,Y)$，概率相加，互斥分类相加）。

| Z | 2 | 3 | 4 | 5 |
|---|-----|-----|-----|-----|
| P | $\frac{1}{4}$ | $\frac{3}{8}$ | … | … |

（最终结果取决于联合分布表格完整值，老师口述了关键概率值）

**例 3.4**（最难）联合密度 $f(x,y) = \begin{cases} \frac{1}{2}e^{-y/2}, & 0<x<1,\; y>0 \\[2pt] 0, & \text{其他} \end{cases}$，求 $Z = X-Y$ 的密度 $f_Z(z)$。

**解**（分布函数法）：

**Step 1 — 定分段点**：区域 $D=\{0<x<1,\;y>0\}$ 的边角点为 $(0,0)$ 和 $(1,0)$。代入 $z=x-y$ 得 $z=0$ 和 $z=1$。分三段讨论：$z<0$，$0\le z<1$，$z\ge 1$。

**Step 2 — 写分布函数**：$F_Z(z) = P(Z\le z) = P(X-Y\le z) = P(Y\ge X-z)$，即求直线 $y=x-z$ 上方与区域 $D$ 交集的二重积分。

**Step 3 — 分区间求 $F_Z(z)$**：

- **$z<0$**（直线在 $y=x$ 上方）：交集为梯形区域 $\{0<x<1,\; x-z<y<+\infty\}$：
  $$F_Z(z) = \int_0^1 \int_{x-z}^{+\infty} \frac{1}{2}e^{-y/2}\,dy\,dx = \int_0^1 e^{-(x-z)/2}\,dx = 2e^{z/2}(1-e^{-1/2})$$

- **$0\le z<1$**（直线穿过 $D$）：用补集法，$F_Z(z)=1\;-$ 下方三角形区域 $\{z<x<1,\;0<y<x-z\}$ 的积分：
  $$F_Z(z) = 1 - \int_z^1 \int_0^{x-z} \frac{1}{2}e^{-y/2}\,dy\,dx$$

- **$z\ge 1$**（直线完全移出 $D$）：整个 $D$ 被覆盖，由规范性得 $F_Z(z)=1$。

**Step 4 — 求导得 $f_Z(z)$**：对上述三段分别求导。$z\ge 1$ 段 $F_Z(z)=1$ 为常数，导数为 $0$。

> 🔑 连续型函数分布核心：**边角点定分段 → 分区间求 $F_Z$ → 求导得 $f_Z$**。

## 第四章 随机变量的数字特征

### 4.1 数学期望 ⭐

| 类型 | 公式 |
|------|------|
| 离散型 | $E(X) = \sum x_i p_i$ |
| 连续型 | $E(X) = \int x f(x)\,dx$ |

**函数期望**：$E[g(X)] = \sum g(x_i)p_i$ 或 $\int g(x)f(x)dx$；$E[g(X,Y)] = \iint g(x,y)f(x,y)dxdy$

**性质**：$E(C)=C$；$E(CX)=CE(X)$；$E(X+Y)=E(X)+E(Y)$；独立则 $E(XY)=E(X)E(Y)$

### 4.2 方差 ⭐

$$D(X) = E[(X-EX)^2] = E(X^2) - [E(X)]^2$$

> 🔑 **「方差 = 平方的期望 − 期望的平方」**

**性质**：$D(C)=0$；$D(CX)=C^2D(X)$；$D(X \pm Y) = D(X) + D(Y) \pm 2\operatorname{Cov}(X, Y)$（独立时 $\operatorname{Cov}=0$，退化为 $D(X)+D(Y)$）。

### 4.3 常见分布期望与方差

| 分布 | $E(X)$ | $D(X)$ |
|------|:------:|:------:|
| $B(1,p)$ | $p$ | $p(1-p)$ |
| $B(n,p)$ | $np$ | $np(1-p)$ |
| $P(\lambda)$ | $\lambda$ | $\lambda$ |
| $U(a,b)$ | $\frac{a+b}{2}$ | $\frac{(b-a)^2}{12}$ |
| $e(\lambda)$ | $\frac{1}{\lambda}$ | $\frac{1}{\lambda^2}$ |
| $N(\mu,\sigma^2)$ | $\mu$ | $\sigma^2$ |

> 🔑 **平方期望公式**：$E(X^2) = D(X) + [E(X)]^2$

### 4.4 协方差

$$\operatorname{Cov}(X,Y) = E(XY) - E(X)E(Y)$$

**性质**：$\operatorname{Cov}(aX,bY)=ab\operatorname{Cov}(X,Y)$；$\operatorname{Cov}(X_1+X_2, Y)=\operatorname{Cov}(X_1,Y)+\operatorname{Cov}(X_2,Y)$（画两条线拆开）；$\operatorname{Cov}(X,X)=D(X)$；独立 $\implies \operatorname{Cov}=0$。

### 4.5 相关系数

$$\rho_{XY} = \frac{\operatorname{Cov}(X,Y)}{\sqrt{D(X)}\sqrt{D(Y)}}$$

| $\rho$ | 含义 |
|:------:|------|
| $\pm 1$ | 完全线性相关 |
| $0$ | 不相关（无线性关系） |
| 接近 0 | 线性关系弱 |

> 🔑 **性质**：$|\rho| \le 1$ 恒成立。$|\rho|=1 \iff P(Y=aX+b)=1$（$a>0$ 时 $\rho=1$，$a<0$ 时 $\rho=-1$）。
> 🔑 **独立 $\Rightarrow$ 不相关，但不相关 $\nRightarrow$ 独立**（如 $Y=X^2$ 不相关但不独立）。
> ⚠️ **例外**：二维正态分布中，**独立 $\iff$ 不相关**（充要条件！考试常考）。

#### 4.7 矩与协方差矩阵

| 概念 | 定义 | 说明 |
|------|------|------|
| **$k$ 阶原点矩** | $E(X^k)$ | $k=1$ 即期望 $E(X)$ |
| **$k$ 阶中心矩** | $E[(X-EX)^k]$ | $k=2$ 即方差 $D(X)$ |
| **协方差矩阵**（$2\times 2$） | $\begin{pmatrix} D(X) & \operatorname{Cov}(X,Y) \\ \operatorname{Cov}(Y,X) & D(Y) \end{pmatrix}$ | 等价 $\begin{pmatrix} \sigma_1^2 & \rho\sigma_1\sigma_2 \\ \rho\sigma_1\sigma_2 & \sigma_2^2 \end{pmatrix}$ |

### 4.6 切比雪夫不等式

$$P(|X - EX| \ge \varepsilon) \le \frac{D(X)}{\varepsilon^2}$$

等价形式：$P(|X - EX| < \varepsilon) \ge 1 - \dfrac{D(X)}{\varepsilon^2}$

> 🔑 口诀「**大小不一**」：**大**于号/没有 1（原式）；**小**于号/有 1（等价式）。

**例 4.1** 方差性质综合 ⭐

$X_1 \sim U(0,6),\; X_2 \sim N(\mu, 4),\; X_3 \sim P(3)$，三者独立。求 $D(X_1 - 2X_2 + 3X_3)$。

**解**：

**例 4.2** 二项分布求 $E(X^2)$

$X$ 表示 10 次独立射击命中次数，命中率 0.4。求 $E(X^2)$。

**解**：$X \sim B(10, 0.4)$。查表：$E(X)=np=4$，$D(X)=np(1-p)=2.4$。

**例 4.3** 离散型协方差与相关系数 ⭐

已知 $X$、$Y$、$XY$ 的分布律，求 $\operatorname{Cov}(X-Y, Y)$ 和 $\rho_{XY}$。

**解**：
$$\operatorname{Cov}(X-Y, Y) = \operatorname{Cov}(X,Y) + \operatorname{Cov}(-Y,Y) = \operatorname{Cov}(X,Y) - D(Y)$$

计算：$E(XY)=\frac{2}{3}$，$E(X)=\frac{2}{3}$，$E(Y)=1$，$\operatorname{Cov}(X,Y)=\frac{2}{3}-\frac{2}{3}\times 1=0$。
$E(Y^2)=\frac{5}{3}$，$D(Y)=\frac{5}{3}-1=\frac{2}{3}$。
$$\operatorname{Cov}(X-Y, Y) = 0 - \frac{2}{3} = -\frac{2}{3}$$

$$\rho_{XY} = \frac{\operatorname{Cov}(X,Y)}{\sqrt{D(X)D(Y)}} = \frac{0}{\cdots} = 0 \quad \text{（不相关）}$$

> 🔑 协方差分配：画两条线，$\operatorname{Cov}(X\pm Y, Z) = \operatorname{Cov}(X,Z) \pm \operatorname{Cov}(Y,Z)$

**例 4.4** 连续型协方差与相关系数

联合密度 $f(x,y) = \begin{cases} \frac{3}{4}x^2y, & 0\le x\le 2, 0\le y\le 1 \\ 0, & \text{其他} \end{cases}$，求 $\rho_{XY}$。

**解**（连续型口诀：取值×密度再积分）：
$$E(XY) = \iint xy f\,dxdy = \frac{3}{4}\int_0^2\int_0^1 x^3 y^2\,dy\,dx = 1$$
$$E(X) = \frac{3}{4}\int_0^2\int_0^1 x^3 y\,dy\,dx = \frac{3}{2}$$
$$E(Y) = \frac{3}{4}\int_0^2\int_0^1 x^2 y^2\,dy\,dx = \frac{2}{3}$$
$$\operatorname{Cov}(X,Y) = 1 - \frac{3}{2}\times\frac{2}{3} = 0 \implies \rho_{XY} = 0$$

**例 4.5** 切比雪夫不等式

$E(X)=100,\; D(X)=50$。估计 $P(80 < X < 120) \ge \underline{\qquad}$。

**解**：$P(80<X<120) = P(|X-100|<20) \ge 1 - \frac{50}{20^2} = 1 - \frac{1}{8} = \frac{7}{8}$

## 第五章 中心极限定理与抽样分布

> 对应视频：P8（27:38）

### 5.1 中心极限定理 ⭐

**定理**：设 $X_1, X_2, \ldots, X_n$ 独立同分布，期望和方差存在，则 $n$ 充分大时：

$$\sum_{i=1}^n X_i \xrightarrow{\text{approx}} N\!\left(E\left(\sum X_i\right),\; D\left(\sum X_i\right)\right)$$

> 🔑 **"中心"**= 不管原始什么分布，求和后以正态为中心；**"极限"**= $n\to\infty$ 时精确服从正态。

**CLT 题型识别三要素**：题目出现 **(1) $n$ 个变量 (2) 独立 (3) 同分布** → 中心极限定理。

#### 📝 例 5.1：电子元件寿命

某种电子元件寿命服从均值 100h 的**指数分布**。随机抽取 16 只（独立），求寿命总和 > 1920h 的概率。

**解**：设 $X_i \sim \text{Exp}(\lambda)$，$E(X_i)=\frac{1}{\lambda}=100$，$D(X_i)=\frac{1}{\lambda^2}=100^2$。

由 CLT：$\sum X_i \xrightarrow{\text{approx}} N(16\times 100,\; 16\times 100^2) = N(1600,\; 160000)$

标准化：
$$P(\sum X_i > 1920) = 1 - P\!\left(Z \le \frac{1920-1600}{400}\right) = 1 - \Phi(0.8)$$

#### 📝 例 5.2：100 个独立部件

100 个独立部件，每个损坏概率 0.1。求至少 85 个正常的概率。

**解**：设 $X_i$=部件是否损坏（0-1 分布），$p=0.1$。$E(X_i)=0.1$，$D(X_i)=0.09$。
$$\sum X_i \xrightarrow{\text{approx}} N(10,\; 9)$$
"至少 85 正常" $\iff$ 损坏数 $\le 15$：
$$P = P\!\left(Z \le \frac{15-10}{3}\right) = \Phi\!\left(\frac{5}{3}\right)$$

> 💡 正态分布连续型 → 严格不等号改写为 $\le$ 加等号，单点概率为 0。

### 5.2 三大抽样分布

| 分布 | 定义 | 密度曲线 | 上 $\alpha$ 分位点 | 分位点性质 | 期望与方差 |
|:---:|---|:---:|:---:|---|---|
| **$\chi^2(n)$** | $X_i\sim N(0,1)$ i.i.d.，$\chi^2=\sum X_i^2$ | ![[assets/images/chisq_density.png\|150]] | ![[assets/images/chisq_quantile.png\|150]] | $P(\chi^2>\chi^2_\alpha(n))=\alpha$<br>可加性：$\chi^2(n_1)+\chi^2(n_2)=\chi^2(n_1+n_2)$ | $E=n$<br>$D=2n$ |
| **$t(n)$** | $X_1\sim N(0,1),\;X_2\sim\chi^2(n)$ 独立<br>$T=\dfrac{X_1}{\sqrt{X_2/n}}$ | ![[assets/images/t_density.png\|150]] | ![[assets/images/t_quantile.png\|150]] | $P(T>t_\alpha(n))=\alpha$<br>$t_{1-\alpha}(n)=-t_\alpha(n)$（对称）<br>$T^2\sim F(1,n)$ | $E=0\;(n>1)$<br>$D=\dfrac{n}{n-2}\;(n>2)$ |
| **$F(n_1,n_2)$** | $X_1\sim\chi^2(n_1),\;X_2\sim\chi^2(n_2)$ 独立<br>$F=\dfrac{X_1/n_1}{X_2/n_2}$ | ![[assets/images/f_density.png\|150]] | ![[assets/images/f_quantile.png\|150]] | $P(F>F_\alpha(n_1,n_2))=\alpha$<br>$F_{1-\alpha}(n_1,n_2)=\dfrac{1}{F_\alpha(n_2,n_1)}$ | $E=\dfrac{n_2}{n_2-2}\;(n_2>2)$<br>$D=\dfrac{2n_2^2(n_1+n_2-2)}{n_1(n_2-2)^2(n_2-4)}\;(n_2>4)$ |

> 🔑 **口诀**：标准正态平方和 → 卡方；标准正态 ÷ 根号下(卡方/自由度) → t；卡分之卡 → F。$n\to\infty$ 时 $t(n)\to N(0,1)$。

#### 📝 例 5.3：$t$ 分布平方 → $F$ 分布

证明：若 $X \sim t(n)$，则 $X^2 \sim F(1, n)$。

**解**：由 $t$ 分布定义 $X = \frac{X_1}{\sqrt{X_2/n}}$（$X_1\sim N(0,1), X_2\sim\chi^2(n)$ 独立）。
$$X^2 = \frac{X_1^2}{X_2/n} = \frac{X_1^2/1}{X_2/n}$$
$X_1^2 \sim \chi^2(1)$，$\therefore X^2 = \frac{\chi^2(1)/1}{\chi^2(n)/n} \sim F(1, n)$。

#### 📝 例 5.4：构造卡方分布

总体 $X\sim N(0,1)$，样本 $X_1,X_2,X_3,X_4$。令 $Y = c(X_1+X_2)^2 + c(X_3+X_4)^2$，求 $c$ 使 $Y\sim\chi^2$。

**解**：$X_1+X_2\sim N(0,2)$，标准化得 $\frac{X_1+X_2}{\sqrt{2}}\sim N(0,1)$。
$$\left(\frac{X_1+X_2}{\sqrt{2}}\right)^2 + \left(\frac{X_3+X_4}{\sqrt{2}}\right)^2 \sim \chi^2(2)$$
对比 $Y$ 表达式得 $c = \frac{1}{2}$，$Y\sim\chi^2(2)$。

> 🔑 一般正态必须先**标准化**再平方，才能拼出卡方。

## 第七章 参数估计

> 对应视频：P9（38:16）

### 7.1 矩估计

**两步走**：(1) 求 $E(X)$；(2) 令 $\bar{X} = E(X)$，解出 $\hat{\theta}$。

### 7.2 极大似然估计 ⭐

**三步走**：
1. 写似然函数 $L(\theta)$（离散型概率连乘，连续型密度连乘，**取非零段**）
2. 取对数 $\ln L(\theta)$（连乘变连加）
3. 求导 $\frac{d}{d\theta}\ln L=0$，解出 $\hat{\theta}$

### 7.3 估计量优劣准则

| 准则 | 定义 | 口诀 |
|------|------|------|
| **无偏性** | $E(\hat{\theta}) = \theta$ | 线性组合无偏 $\iff$ 系数和为 1 |
| **有效性** | 同为无偏时，方差越小越有效 | — |
| **一致性** | $n$ 越大，$\hat{\theta}$ 越接近 $\theta$ | — |

### 7.4 区间估计

**三大枢轴量**（必须背）：

| 场景 | 枢轴量 | 分布 | 置信区间 |
|------|--------|------|----------|
| $\sigma^2$ 已知，求 $\mu$ | $\dfrac{\bar{X}-\mu}{\sigma/\sqrt{n}}$ | $N(0,1)$ | $\left[\bar{X} - u_{\alpha/2}\frac{\sigma}{\sqrt{n}},\; \bar{X} + u_{\alpha/2}\frac{\sigma}{\sqrt{n}}\right]$ |
| $\sigma^2$ 未知，求 $\mu$ | $\dfrac{\bar{X}-\mu}{s/\sqrt{n}}$ | $t(n-1)$ | $\left[\bar{X} - t_{\alpha/2}(n-1)\frac{s}{\sqrt{n}},\; \bar{X} + t_{\alpha/2}(n-1)\frac{s}{\sqrt{n}}\right]$ |
| $\mu$ 未知，求 $\sigma^2$ | $\dfrac{(n-1)S^2}{\sigma^2}$ | $\chi^2(n-1)$ | $\left[\dfrac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)},\; \dfrac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}\right]$ |

> ⚠️ $t$ 分布自由度是 **$n-1$** 不是 $n$（考试常挖坑）；$\chi^2$ 不对称，左右分位点不同。

#### 📝 第七章典型例题

**例 7.1** 总体密度 $f(x;\theta)=(\theta+1)x^\theta,\;0<x<1$。求 $\theta$ 的矩估计量和极大似然估计量。

**解（矩估计）**：
$$E(X) = (\theta+1)\int_0^1 x^{\theta+1}dx = \frac{\theta+1}{\theta+2} = \bar{X} \implies \hat{\theta} = \frac{2\bar{X}-1}{1-\bar{X}}$$

**解（MLE）**：
$$L(\theta) = (\theta+1)^n \left(\prod x_i\right)^\theta$$
$$\ln L = n\ln(\theta+1) + \theta\sum\ln x_i$$
$$\frac{d}{d\theta}\ln L = \frac{n}{\theta+1} + \sum\ln x_i = 0 \implies \hat{\theta} = -\frac{n}{\sum\ln X_i} - 1$$

**例 7.2** $X_1,X_2\sim N(\mu,1)$，$\hat{\mu}_1=\frac{1}{3}X_1+\frac{2}{3}X_2,\;\hat{\mu}_2=\frac{1}{4}X_1+\frac{3}{4}X_2$。判断无偏性和有效性。

**解**：$E(\hat{\mu}_1)=\frac{1}{3}\mu+\frac{2}{3}\mu=\mu$ ✓；$E(\hat{\mu}_2)=\frac{1}{4}\mu+\frac{3}{4}\mu=\mu$ ✓。两者均无偏。
$$D(\hat{\mu}_1)=\frac{5}{9},\; D(\hat{\mu}_2)=\frac{5}{8}\; \implies \hat{\mu}_1 \text{ 更有效}$$

> 🔑 系数和为 1 → 无偏（秒杀选择题）。

**例 7.3** 直径 $X\sim N(\mu,\sigma^2)$，$n=6,\;\bar{x}=14.9,\;s=0.0226$，置信度 95%。求 $\mu$ 和 $\sigma^2$ 的置信区间。

**解（$\mu$，$\sigma^2$ 未知）**：用 $t$ 分布，$\bar{x} \pm \frac{s}{\sqrt{n}} \cdot t_{\alpha/2}(n-1)$。自由度 $=5$。

**解（$\sigma^2$，$\mu$ 未知）**：用 $\chi^2$ 分布，
$$\left[ \frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)},\; \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)} \right]$$

## 第八章 假设检验

> 对应视频：P10（13:45）

### 8.1 基本概念

- **$H_0$（原假设）**：$\mu=\mu_0$，代表"正常/达标"，不能轻易推翻
- **$H_1$（备择假设）**：$\mu\neq\mu_0$，代表"不正常"
- **小概率原理**：小概率事件在一次实验中基本不可能发生，如果发生了就拒绝 $H_0$

### 8.2 假设检验四步法

1. **提出假设**：写出 $H_0$ 和 $H_1$
2. **选统计量**：在 $H_0$ 成立下构造
3. **定拒绝域**：小概率 $\alpha$ 对应区间
4. **做判断**：代入样本，看是否落入拒绝域

### 8.3 三种检验法

| 检验法 | 对象 | 条件 | 拒绝域（双边 $H_1:\neq$） | 拒绝域（右侧 $H_1:>$） | 拒绝域（左侧 $H_1:<$） |
|--------|------|------|--------------------------|----------------------|----------------------|
| **U 检验** | $\mu$ | $\sigma^2$ 已知 | $(-\infty,-u_{\alpha/2})\cup(u_{\alpha/2},+\infty)$ | $U > u_\alpha$ | $U < -u_\alpha$ |
| **T 检验** | $\mu$ | $\sigma^2$ 未知 | $(-\infty,-t_{\alpha/2})\cup(t_{\alpha/2},+\infty)$ | $T > t_\alpha(n-1)$ | $T < -t_\alpha(n-1)$ |
| **$\chi^2$ 检验** | $\sigma^2$ | 正态总体 | $(0,\chi^2_{1-\alpha/2})\cup(\chi^2_{\alpha/2},+\infty)$ | $\chi^2 > \chi^2_\alpha(n-1)$ | $\chi^2 < \chi^2_{1-\alpha}(n-1)$ |

> 🔑 $u_{0.025}=1.96$（必记）。**单边用 $\alpha$，双边用 $\alpha/2$**。

> 🔑 $u_{0.025}=1.96$（必记）。

---

#### 📝 第八章典型例题

**例 8.1（U 检验）** 食盐包装机，正常时 $\mu=500$g，$\sigma^2=15^2$。抽 9 袋得 $\bar{x}=511.22$，$\alpha=0.05$，问机器是否正常？

**解**：$H_0:\mu=500,\;H_1:\mu\neq 500$。
$$U = \frac{511.22-500}{15/3} = 2.244 > 1.96 = u_{0.025}$$
拒绝 $H_0$，机器不正常。

**例 8.2（$\chi^2$ 检验）** 饮料方差要求 $\sigma_0^2=0.0004$。抽 20 袋得 $s=0.03$，$\alpha=0.05$，是否达标？

**解**：$H_0:\sigma^2=0.0004,\;H_1:\sigma^2\neq 0.0004$。
$$\chi^2 = \frac{19\times 0.03^2}{0.0004} = 42.75$$
远大于 $\chi^2_{0.025}(19)$，拒绝 $H_0$，未达标。

## 附录：常见分布速查表

### 离散型

| 分布 | 分布律 | $E(X)$ | $D(X)$ |
|------|--------|:------:|:------:|
| 0-1 $B(1,p)$ | $p^k(1-p)^{1-k}$ | $p$ | $p(1-p)$ |
| 二项 $B(n,p)$ | $C_n^k p^k (1-p)^{n-k}$ | $np$ | $np(1-p)$ |
| 泊松 $P(\lambda)$ | $\frac{\lambda^k}{k!}e^{-\lambda}$ | $\lambda$ | $\lambda$ |
| 几何 $G(p)$ | $(1-p)^{k-1}p$ | $\frac{1}{p}$ | $\frac{1-p}{p^2}$ |

### 连续型

| 分布                   | 密度 $f(x)$                                                     |       $E(X)$        |        $D(X)$         |
| -------------------- | ------------------------------------------------------------- | :-----------------: | :-------------------: |
| 均匀 $U(a,b)$          | $\frac{1}{b-a}$                                               |   $\frac{a+b}{2}$   | $\frac{(b-a)^2}{12}$  |
| 指数 $e(\lambda)$ | $\lambda e^{-\lambda x}$ | $\frac{1}{\lambda}$ | $\frac{1}{\lambda^2}$ |
| 正态 $N(\mu,\sigma^2)$ | $\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ |        $\mu$        |      $\sigma^2$       |

### 核心公式速记

| 场景 | 公式 |
|------|------|
| 标准化 | $Z = \frac{X-\mu}{\sigma} \sim N(0,1)$ |
| 对称性 | $\Phi(-x) = 1 - \Phi(x)$ |
| 方差 | $D(X) = E(X^2) - E^2(X)$ |
| 平方期望 | $E(X^2) = D(X) + E^2(X)$ |

| 切比雪夫 | $P(\lvert X-EX\rvert \ge\varepsilon) \le \frac{DX}{\varepsilon^2}$ |
