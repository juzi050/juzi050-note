# 一、授课范围与重点

| 章节                         | 授课 / 考核情况                                                                                                                                        | 重点内容                                                                                                                                                         |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 第 1 章 概论                   | 1.1 计算机的产生与发展：可以简单介绍，或者让学生自学。                                                                                                                    | 计算机硬件系统（五大部件）  <br/>冯·诺依曼体系结构特点  <br/>计算机三种语言  <br/>计算机系统层次结构  <br/>计算机性能指标计算（可与 7.1.4 节结合，期末考试）                                                             |
| 第 2 章 总线                   | 大纲中没有安排时间，可以全部不讲；在第 4 章和第 7 章涉及总线时，可简单说明总线的功能、特点（分时共享）、类型（内总线、系统总线、I/O 总线等）。2.2 节总线结构中南北桥、前端总线、PCH 概念，可在第 7 章讲解时简单讲下演变过程。                        |                                                                                                                                                              |
| 第 3 章 信息编码与数据表示            | 3.1 数制：不讲，前导课已讲过，考试一定会用到。3.5 非数值数据的表示：可讲解，也可让学生自学。3.6.3 海明校验码：不讲不考。                                                                              | 定点数表示（原码、补码、移码）  <br/>浮点数规格化表示（自定义格式、IEEE754 标准）  <br/>计算机高级语言中数据类型转换涉及的编码（考试会考）                                                                             |
| 第 4 章 运算方法与运算器             | 4.2.3 十进制加法运算：不讲，数电已经讲过。4.3 定点数的乘法运算：讲原理，计算不要求；其中 4.3.3 补码串行乘法器可不讲（但 408 统考有考到）。4.4 定点数的除法运算：讲原理，计算不要求。4.7 现代计算机的运算器：简介。                         | 补码加减运算、移位运算  <br/>运算器的结构、数据通路  <br/>标志寄存器  <br/>浮点数的加减法运算                                                                                                    |
| 第 5 章 存储系统                 | 图片中未显示单独删减说明，按右侧重点内容复习。                                                                                                                          | 存储器的分类、层次结构  <br/>多字节数据在存储器中的存放顺序：大小端  <br/>主存储器 SRAM、ROM 的扩展设计  <br/>多体交叉存储器、相联存储器  <br/>Cache 的原理、地址映射、命中率  <br/>虚拟存储器的概念、原理（页式、段式、段页式）                    |
| 第 6 章 指令系统                 | 6.5 指令系统举例：不讲。6.6 RISC-V 指令系统和 6.7 ARM 指令系统：根据教师选择的模型机结构选讲一个（绝大多数教师讲 RISC-V；如果讲 ARM，需要自己出题）。6.6 RISC-V 指令系统有完整视频，建议让学生自学 2 周，然后再上 1-2 节例题 + 习题课。 | 指令格式的分析与设计（包括指令操作码扩展技术）  <br/>指令的寻址方式（指令寻址和数据寻址）  <br/>针对特定指令系统：机器指令的编码和解码、执行结果（汇编语言与机器语言之间的转换）  <br/>针对 RISC-V 或者 ARM：从高级语言程序到机器语言的转换                       |
| 第 7 章 控制器                  |                                                                                                                                                  | 控制器的功能和组成  <br/>硬布线控制器的原理与设计  <br/>单周期、多周期与流水线实现的性能对比  <br/>基于 Yy-202 模型机或其他模型机的系统结构，进行指令执行过程的分析（已知执行过程，分析指令功能）与设计（已知指令功能，分析执行过程）  <br/>微程序控制器的基本原理、微指令格式的设计 |
| 第 8 章 RISC-V 与 ARM 模型机设计实例 | 根据教师选择的模型机结构，选讲 8.1 RISC-V 模型机的设计或者 8.2 ARM 模型机的设计（1 次课 / 3 学时）。如果没有时间，可以安排到实验课去讲。                                                               | RISC-V：  <br/>运算及传送指令的数据通路设计：对应实验 8 的原理  <br/>访存指令的数据通路设计：对应实验 9 的原理  <br/>转移类指令的数据通路设计：对应实验 10 的原理  <br/>RISC-V 模型机控制单元 CU 的设计：可以结合实验 8 讲有限状态机              |
| 第 9 章 输入输出系统               |                                                                                                                                                  | 外设的编址方式、I/O 接口的功能和组成  <br/>主机与外设交换信息的方式（四种方式）  <br/>中断系统：中断概念及过程、中断分类、中断屏蔽、中断响应、中断嵌套、中断判优、中断向量等概念                                                            |

# 4.计组实验复习重点

| 序号  | 模块         | 学习重点                                                                                        |
| --- | ---------- | ------------------------------------------------------------------------------------------- |
| 1   | RISC-V基础   | 掌握ISA特点：模块化设计、32位定长指令、32个通用寄存器、立即数扩展、大小端、寻址方式。重点理解 `addi、andi、lui、sll、sw、jal、jalr` 等常见指令功能。 |
| 2   | 指令编码       | 熟悉R/I/S/U等格式，能根据机器码拆分 `opcode、rd、funct3、rs1、rs2、imm、funct7`，并反推汇编指令。                        |
| 3   | 汇编程序分析     | 能读懂简单汇编片段，判断寄存器变化、内存访问、循环、赋值、移位和存储等程序功能。                                                    |
| 4   | C转汇编       | 重点练习赋值语句、加减运算、数组访问、左移、内存存取等C语言片段对应汇编写法。                                                     |
| 5   | CPU设计      | 重点掌握取指、译码、执行、访存、写回流程；会画执行流程图、有限状态机，并写出各状态控制信号。                                              |
| 6   | Verilog与测试 | 复习系统结构图中关键模块，如PC、IR、寄存器堆、ALU、存储器、立即数扩展器和MUX。能写测试程序机器码，并用寄存器/内存结果判断执行是否正确。                   |
| 7   | 拓展题        | 理解控制信号故障影响、冗余部件删除、用ALU替代PC+4或分支加法器后的结构和流程变化。                                                |
