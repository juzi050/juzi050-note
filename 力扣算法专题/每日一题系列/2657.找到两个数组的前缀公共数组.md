[2657. 找到两个数组的前缀公共数组 - 力扣（LeetCode）](https://leetcode.cn/problems/find-the-prefix-common-array-of-two-arrays/description/?envType=daily-question&envId=2026-05-20)

条件约束为：

- A与B数组都为长度为n的排列
- C[i]=A[0\~i]和B[0\~i]的公共元素个数

初始思路：

- 采用数组记录A和B的0\~i的公共元素个数

```Java
class Solution {
    public int[] findThePrefixCommonArray(int[] A, int[] B) {
        int n=A.length;
        int[] count=new int[n+1];
        int[] C=new int [n];
        int temp=0;
        for(int i=0;i<n;i++){
            int a=A[i];
            int b=B[i];
            if(a==b){
                temp++;
            }else{
                count[a]++;
                count[b]--;
                temp+=count[a]==0?1:0;
                temp+=count[b]==0?1:0;
            }
            C[i]=temp;
        }
        return C;
    }
}
```

优化思路：

- 采用位运算记录A和B的0\~i的公共部分

```Java
class Solution {
    public int[] findThePrefixCommonArray(int[] A, int[] B) {
        int n=A.length;
        int[] C=new int [n];
        long a=0,b=0;
        for(int i=0;i<n;i++){
            a|=1L<<A[i];
            b|=1L<<B[i];
            C[i]=Long.bitCount(a&b);
        }
        return C;
    }
}
```
