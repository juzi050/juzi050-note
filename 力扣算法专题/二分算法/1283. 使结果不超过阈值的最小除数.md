[1283. 使结果不超过阈值的最小除数](https://leetcode.cn/problems/find-the-smallest-divisor-given-a-threshold/)
**思路**：
1. 看C++语言，返回的是INT类型，r直接根据提交记录试（推荐直接取最大值，运行速度慢，但能过），都是正整数，l取0。遵循左开右开写法。
2. 以threshold为分界线
3. $temp>threshold$ 代表答案在右半区间，不然答案在左半区间
```typescript
function smallestDivisor(nums: number[], threshold: number): number {

    let l=0,r=1e6+7,mid=0,temp=0;

    while(l+1<r){

        mid=l+((r-l)>>1);

        //temp=nums.reduce((a,b)=>a+Math.ceil(b/mid),0);

        temp=0;

        for(let i=0;i<nums.length&&temp<=threshold;i++){

            temp+=((nums[i]+mid-1)/mid|0);

        }

        if(temp>threshold){

            l=mid;

        }else{

            r=mid;

        }

    }

    return r;

};
```
