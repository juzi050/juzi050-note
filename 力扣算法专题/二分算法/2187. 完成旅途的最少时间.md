[2187. 完成旅途的最少时间 - 力扣（LeetCode）](https://leetcode.cn/problems/minimum-time-to-complete-trips/)
**思路**：
1. 题目翻译为数学语言：$$\sum_{i=0}^{n-1}\lfloor{{\frac{ans}{time}}  \rfloor \geq totalTrips}$$
2. 采用左右开区间的二分模板$[r,maxInt]能满足上述条件，最后返回r$
```ts
function minimumTime(time: number[], totalTrips: number): number {

    let l = 0;

    const UPPER = 1000000000000000;

    let r = UPPER;

    let sum,mid;

    while(l+1<r){

        mid = Math.floor(l + (r - l) / 2);

        sum=0;

        for(let i=0;i<time.length&&sum<totalTrips;i++){

            sum+=Math.floor(mid/time[i]);

        }

        if(sum>=totalTrips){//时间充裕

            r=mid;

        }else{

            l=mid;

        }

    }

    return r;

};
```
