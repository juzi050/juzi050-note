# 2875. 无限数组的最短子数组

https://leetcode.cn/problems/minimum-size-subarray-in-infinite-array/description/

思路：

1. 先把target变成<$\sum_{i=0}^n \mathit{nums}_i$
2. 问题转化为求任意一点开始的最短的=target的子数组

   1. 第一种求法，从头遍历两遍到尾
   2. 同时记录子数组的最大值和最小值

```TypeScript
function minSizeSubarray(nums: number[], target: number): number {
    let sum = nums.reduce((a,b)=> a+b, 0);
    let temp = Math.floor(target / sum);
    target = target % sum;
    let ans = nums.length+1;
    let left = 0,s1=0,n=nums.length;
    for(let r = 0;r<2*nums.length;r++){
        s1+=nums[r%n];
        while(s1>target){
            s1-=nums[left%n];
            left++;
        }
        if(s1===target){
            ans=Math.min(ans,r-left+1);
        }
    }
    if(ans!=n+1){
        ans+=temp*n;
    }else{
        ans=-1;
    }
    return ans;
};
```
