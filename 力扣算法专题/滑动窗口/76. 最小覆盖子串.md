# 76. 最小覆盖子串

https://leetcode.cn/problems/minimum-window-substring/description/

思路：

- 用cnt数组维护t中字符出现的次数
- 右侧进，cnt[t]--
- 当cnt[s[l]]<0,l++

```TypeScript
function minWindow(s: string, t: string): string {
    let ans = "";
    let cnt= new Array<number>(256).fill(0);
    for(let c of t){
        cnt[c.charCodeAt(0)]++;
    }
    let l=0,r=0;
    for(;r<s.length;r++){
        cnt[s[r].charCodeAt(0)]--;
        while(l<r&&cnt[s[l].charCodeAt(0)]<0){
            cnt[s[l].charCodeAt(0)]++;
            l++;
        }
        let flag=true;
        cnt.forEach(v=>flag=(flag&&(v<=0)));
        if(flag){
            if(ans===""){
                ans=s.substring(l,r+1);
            }else if(r-l+1<ans.length){
                ans=s.substring(l,r+1);
            }
        }
    }
    return ans;
};
```
