[1208. 尽可能使字符串相等 - 力扣（LeetCode）](https://leetcode.cn/problems/get-equal-substrings-within-budget/description/)

条件约束为：

- 对于区间 ⁍，定义 ⁍
- 在 ⁍ 的条件下，求 ⁍ 的最大值

思路：

- 不定长滑动窗口
- 当 ⁍ 时，⁍

```TypeScript
class Solution {
    public int equalSubstring(String S, String T, int maxCost) {
        int l=0;
        int res=0;
        char[] s=S.toCharArray(),t=T.toCharArray();
        int n=s.length;
        int cost=0;
        for(int i=0;i<n;i++){
            cost+=Math.abs(s[i]-t[i]);
            while(cost>maxCost){
                cost-=Math.abs(s[l]-t[l]);
                l++;
            }
            res=Math.max(i-l+1,res);
        }
        return res;
    }
}
```
