[1695. 删除子数组的最大得分 - 力扣（LeetCode）](https://leetcode.cn/problems/maximum-erasure-value/description/)

条件约束为：

- 删除的子数组[l,r]不能有重复元素
- 子数组[l,r]的max
- 数组元素均大于0

思路：

- 滑动窗口
- 用HashSet维护子数组的元素
- 由于数组元素均大于0，则越多越好
- 碰到重复元素的时候，l++

```
class Solution {
    public int maximumUniqueSubarray(int[] nums) {
        int res=0;
        int l=0;
        Set<Integer> number = new HashSet<>();
        int temp=0;
        for(int r=0;r<nums.length;r++){
            while(number.contains(nums[r])){
                temp-=nums[l];
                number.remove(nums[l++]);
            }
            temp+=nums[r];
            number.add(nums[r]);
            res=Math.max(res,temp);
        }
        return res;
    }
}
```
