[2904. 最短且字典序最小的美丽子字符串 - 力扣（LeetCode）](https://leetcode.cn/problems/shortest-and-lexicographically-smallest-beautiful-string/description/)

条件约束为：

- 求满足count(1)=k的最短字符串，在最短基础上并且是最小的

思路：

- 滑动窗口
- 用cnt记录1的数量
- 出的条件为cnt>k或者s[l]===’0’，因为最短开头肯定是1开头。

```TypeScript
function shortestBeautifulSubstring(s: string, k: number): string {
    let cnt=0,l=0;
    let res="";
    for(let i=0;i<s.length;i++){
        if(s[i]==='1'){
            cnt++;
        }
        while(cnt>k||s[l]==='0'){
            if(s[l]==='1'){
                cnt--;
            }
            l++;
        }
        if(cnt===k){
            let temp=s.substring(l,i+1);
            if(res.length===0){
                res=temp;
            }else if(res.length>temp.length){
                res=temp;
            }else if(res.length===temp.length&&res>temp){
                res=temp;
            }
        }
    }
    return res;
};
```
