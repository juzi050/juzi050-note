[904. 水果成篮 - 力扣（LeetCode）](https://leetcode.cn/problems/fruit-into-baskets/description/)

条件约束为：

- [l,r]之间最多只有两个数
- 求r-l+1的max

思路：

- 滑动窗口
- 用hash表记录[l,r]的数字数量
- 当hash.size()>2的时候，l++

```Java
class Solution {
    public int totalFruit(int[] fruits) {
        int l=0;
        Map<Integer,Integer> count = new HashMap<>();
        int res=0;
        for(int r=0;r<fruits.length;r++){
            count.merge(fruits[r],1,Integer::sum);
            while(count.size()>2){
                int lc=fruits[l++];
                count.merge(lc,-1,Integer::sum);
                if(count.get(lc)==0){
                    count.remove(lc);
                }
            }
            res=Math.max(res,r-l+1);
        }
        return res;
    }
}
```
