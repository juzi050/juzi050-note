# 2024.考试的最大困扰度

[2024. 考试的最大困扰度 - 力扣（LeetCode）](https://leetcode.cn/problems/maximize-the-confusion-of-an-exam/description/)

条件约束为：

- [l,r]之间的T和F的个数不能同时超过k个

思路：

- 滑动窗口
- 用两个count分别记录T和F的个数
- 当countT>k&&countF>k的时候，l++

```Java
class Solution {
    public int maxConsecutiveAnswers(String AnswerKey, int k) {
        char[] answerKey =AnswerKey.toCharArray();
        int l=0,ct=0,cf=0,res=0;
        for(int r=0;r<answerKey.length;r++){
            if(answerKey[r]=='T'){
                ct++;
            }else{
                cf++;
            }
            while(ct>k&&cf>k){
                char out=answerKey[l++];
                if(out=='T'){
                    ct--;
                }else{
                    cf--;
                }
            }
            res=Math.max(r-l+1,res);
        }
        return res;
    }
}
```
