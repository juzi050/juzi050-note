# 1234.替换子串得到平衡字符串

[1234. 替换子串得到平衡字符串 - 力扣（LeetCode）](https://leetcode.cn/problems/replace-the-substring-for-balanced-string/description/)

条件约束为：

- 长度为4m的字符串，只有四个字符
- 替换其中的一个子串，让每个字符出现的次数为m

思路：

- 用一个map记录该字符差m多少
- 右边进->map对应的字符c--
- 左边出的条件为map(s[l])<0 && l<=r
- 结果记录条件为map里所有字符<0

```TypeScript
function balancedString(s: string): number {
    let arr = new Map<string,number>();
    let cnt = new Map<string,number>();
    for(let c of s){
        arr.set(c,(arr.get(c)??0)+1);
    }
    let n = s.length;
    for(const [key,value] of arr){
        cnt.set(key,value-n/4);
    }
    let l=0,r=0,res=n;
    for(;r<n;r++){
        const c = s[r];
        cnt.set(c,cnt.get(c)-1);
        while(cnt.get(s[l])<0&&l<=r){
            const left = s[l++];
            cnt.set(left,cnt.get(left)+1);
        }
        let flag = true;
        for(const [key,value] of cnt){
            if(value>0){
                flag=false;
                break;
            }
        }
        if(flag){
            res=Math.min(res,r-l+1);
        }
    }
    return res;
};
```
