# 2958.最多K个重复元素的最长子数组

[2958. 最多 K 个重复元素的最长子数组 - 力扣（LeetCode）](https://leetcode.cn/problems/length-of-longest-subarray-with-at-most-k-frequency/description/)

条件约束为：

- [l,r]之间最多只有K个重复元素
- 求r-l+1的max

思路：

- 滑动窗口
- 用HashMap维护子数组的元素
- 当新加入的这个元素次数>k的时候，l++

```Java
class Solution {
    public int maxSubarrayLength(int[] nums, int k) {
        int l=0;
        Map<Integer,Integer> count = new HashMap<>();
        int res=0;
        for(int i=0;i<nums.length;i++){
            count.merge(nums[i],1,Integer::sum);
            while(count.get(nums[i])>k){
                count.merge(nums[l++],-1,Integer::sum);
            }
            res=Math.max(res,i-l+1);
        }
        return res;
    }
}
```
